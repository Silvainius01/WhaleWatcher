# Whale Watcher

Whale Watcher is a custom stat tracker for PlanetSide 2 primarily intended for providing data for streaming overlays by transmitting JSON over a websocket connection.

Built using:
- [NetMQ](https://github.com/zeromq/netmq)
- [DaybreakGames.Cenus](https://github.com/Voidwell/DaybreakGames.Census) by Lampjaw
- [CommandEngine](https://github.com/Silvainius01/CommandEngine) by some guy.
